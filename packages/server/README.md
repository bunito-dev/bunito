# `@bunito/server`

Bun server integration for bunito.

This package provides the lower-level server module, request and websocket
contexts, server extensions, and transport exceptions used by higher-level
packages such as `@bunito/http`.

## Installation

```bash
bun add @bunito/server
```

## Usage

```ts
import { HttpException, RequestContext } from '@bunito/server';
```
