# `@bunito/http`

HTTP module for bunito applications.

It provides controllers, route decorators, request injections, middleware, JSON
handling, and HTTP exceptions.

## Installation

```bash
bun add @bunito/http
```

## Usage

```ts
import {
  Controller,
  Get,
  HTTPModule,
  JSONMiddleware,
  JSONModule,
  Post,
  UseMiddleware,
} from '@bunito/http';
```
