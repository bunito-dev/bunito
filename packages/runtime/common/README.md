# @bunito/common

Internal package with small shared building blocks used by the rest of bunito.

This package currently holds low-level helpers, metadata utilities, and base exceptions.
It is mainly consumed by `@bunito/core` and `@bunito/http`, not intended as the main starting point for application code.

If you are getting started with bunito, begin with:

- [`@bunito/core`](../core/README.md)
- [`@bunito/http`](../http/README.md)
- [`docs/getting-started.md`](../../docs/getting-started.md)
